# Hull Tactical – Market Prediction

## Capstone Project: Imperial College London Professional Certificate in Machine Learning and AI

This project focuses on predicting S&P 500 returns using financial time-series data as part of the Hull Tactical Kaggle competition.

## Competition Overview

**Hull Tactical** challenges participants to predict market returns using machine learning techniques applied to financial indicators, macroeconomic data, and technical signals.

## Key Topics Covered

- **Module 6:** Execution plan and project management strategy
- **Module 7:** Cross-validation techniques for time-series data (forward-chaining, expanding window)
- **Module 8:** K-Nearest Neighbors applications and hyperparameter tuning
- **Module 9:** Decision trees for feature importance and hyperparameter optimization
- **Module 10:** Bagging, ensemble methods, fairness/interpretability, and KNN vs Decision Tree comparison
