This repository contains my coursework and capstone project for the **Professional Certificate in Machine Learning and Artificial Intelligence** offered by [Imperial College Business School](https://www.imperial.ac.uk/business-school/executive-education/technology-analytics-data-science/professional-certificate-machine-learning-and-artificial-intelligence-programme/online/), developed jointly with the Department of Computing.

## Capstone Project: Hull Tactical - Market Prediction

My capstone project focuses on the **Hull Tactical S&P 500 Market Return Prediction** challenge, applying machine learning techniques to predict market movements using financial indicators.

### Problem Statement
Predict S&P 500 returns using a rich set of financial indicators including volatility measures, interest rates, momentum indicators, and economic signals.

### Approach
- **Exploratory Data Analysis** with feature correlation and importance analysis
- **Model Development** using KNN, Decision Trees, Random Forests, Naive Bayes, Logistic Regression, and SVMs
- **Hyperparameter Tuning** via cross-validation and Bayesian optimisation
- **Performance Evaluation** using forward-chaining validation for time-series data

### Black-Box Optimisation Challenge
Parallel to the main prediction task, I participated in a **Black-Box Optimisation (BBO)** challenge — maximising 8 unknown functions (2D to 8D) with limited query budgets. This mirrors real-world scenarios like hyperparameter tuning where function evaluations are expensive.

**Key Techniques Applied:**
- Gaussian Process surrogate models
- Acquisition functions (UCB, Expected Improvement)
- Exploration-exploitation trade-off management
- SVM-based region classification for high-dimensional search
